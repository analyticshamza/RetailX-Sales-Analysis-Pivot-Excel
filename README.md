## Project Overview
This project analyzes RetailX sales data using Microsoft Excel Pivot Tables to understand sales performance across channels, time periods, and regions.

The analysis focuses on summarizing total sales, identifying trends by month and quarter, and calculating average sale value per item.

---

## Objectives
- Analyze total sales by channel (In-store vs Online)
- Identify monthly and quarterly sales trends
- Calculate average sale per item using a calculated field
- Enable interactive filtering using slicers

---

## Dataset
- RetailX Sales Dataset
- Key fields include:
  - Date
  - Region
  - Channel
  - Quantity
  - Total Sales

---

## Tools Used
- Microsoft Excel

---

## Analysis Performed

### 1. Pivot Table Creation
- Created pivot table with:
  - Rows: Quarter and Month
  - Columns: Sales Channel
  - Values: Total Sales and Average Sale per Item

---

### 2. Date Grouping
- Grouped sales data by:
  - Quarters
  - Months
- Enabled time-based trend analysis

---

### 3. Calculated Field
- Created a calculated field:
Avg Sale per Item = Total Sales / Quantity
- Aggregated using average to represent typical item price

---

### 4. Slicers
- Added slicers for:
- Region
- Channel
- Enabled interactive filtering and comparison

---

## Key Insights
- Online sales contribute higher total revenue than In-store sales
- Q2 and Q3 show stronger sales performance
- Average sale per item remains relatively stable across channels
- Regional filtering reveals differences in sales contribution
